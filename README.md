# sjournal
A simple and lightweight notepad for the command line.

## Installation
### via pip
```bash
> git clone https://github.com/SamuelStuver/sjournal.git
> cd sjournal
> pip install -r requirements.txt --user
```
### via pipenv
```bash
> git clone https://github.com/SamuelStuver/sjournal.git
> cd sjournal
> pipenv install
```
